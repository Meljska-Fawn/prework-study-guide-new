HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML attribute defines additional information about an element.
  • Inside the head element we use metadata tags for the title, style, base, link, meta, and script.
  • Child elements are elements that are nested inside other elements.
  • Parent elements are elements that hold all the children.
